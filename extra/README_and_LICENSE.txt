#### README
RuneAMNを使ったフリー素材集です。
不具合・改善要望などありましたら作者に連絡いただければ幸いです。

## フリー英文
有名な"Lorem Ipsum"です。
Webサービス上で自動生成したものですが、サイト管理者に確認しましたので、ライセンスはクリアになっています。
長文が必要だけれど、探したり書く時間がない、といった場合にお使いください。

## 魔法陣
project daisy bellが作成した、完全オリジナルかつフリーな魔法陣素材です。
#".svg"はフリースケール(可変サイズ)のベクタ画像素材です。そのまま使う場合など、主にこのファイルをお使いください。
#".jpg"はサムネイル画像です。サインなどが入っているので画像素材には向きませんが、もちろん使用を禁止してはいません。
#".ai"は元素材です。ルーンの文面を変えたい場合や、デザイン変更して使いたい場合にお使いください。

ai/svg画像は、一部アプリケーションで互換性に難がある場合があります。報告いただければ対応を考えます。
AI画像および一部SVG画像は、RuneAMNシリーズフォントがインストールされていない場合、代替フォントに置き換わるため正常に表示されません。
